# 🌿 Sandbox AI

**Express your emotions without words.**

Sandbox AI is an emotional exploration platform that lets users express how they feel by placing objects in a virtual 3D sandbox — no words needed. An AI then reads the arrangement and delivers a gentle, meaningful reflection.

---

## ✨ Features

- **3D Sandbox Experience** — Drag and place objects (people, nature, buildings, animals) in a 3D scene using your instinct
- **AI-Powered Reflections** — Submit your scene and receive a personalized emotional insight generated by AI
- **Emotional Dashboard** — Track your emotional patterns over time with charts and AI-generated insights
- **Camera & Drag Modes** — Switch between placing objects and freely orbiting the sandbox
- **Token-Based System** — Purchase tokens to unlock additional AI reflections and premium features
- **Calm, Immersive Design** — Soft nature-inspired visuals, scroll-based animations, and a peaceful UI throughout

## 🚀 Getting Started

### Prerequisites

- **Node.js** ≥ 18
- **npm**, **bun**, or **pnpm**

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/sandbox-ai.git
cd sandbox-ai

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.

### Build for Production

```bash
npm run build
npm run preview
```

## 🎨 Design System

The project uses a nature-inspired design system with semantic HSL color tokens defined in `src/index.css` and extended via `tailwind.config.ts`. All components use design tokens (`--primary`, `--background`, `--card`, etc.) for consistent theming.

- **Font**: Custom display + body font pairing
- **Palette**: Earthy greens, soft neutrals, calming tones
- **Components**: Built on [shadcn/ui](https://ui.shadcn.com/) with custom variants
